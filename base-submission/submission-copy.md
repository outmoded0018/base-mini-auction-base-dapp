# Base Mini Auction

## App Name

Base Mini Auction

## Tagline

Bid on Base fast

## Description

Create a compact auction on Base, accept live bids, and settle the winner with a clear onchain record.

Character count: 103

## Category

Marketplace / auction / onchain utility

## Short Pitch

Base Mini Auction gives users a fast way to list a small item, place competitive bids, and settle a clear winner directly on Base.

## Production Environment Variables

- `NEXT_PUBLIC_MINI_AUCTION_CONTRACT_ADDRESS`: deployed auction contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
